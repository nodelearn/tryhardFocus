# tryhardFocus

A ideia de criar este site surgiu de uma necessidade genuína minha: ter algo prático e simples para monitorar meu tempo de foco.
Nunca gostei muito dos aplicativos e sites disponíveis para isso, então decidi fazer o meu próprio. Foi uma experiência bem interessante adorei dividir o problema em partes menores e ir pesquisando cada conceito que eu precisava na hora H.

Estou aprendendo, e essa foi uma experiência muito prazerosa. Tenho certeza de que minha curiosidade ainda vai me levar muito longe.

Obrigado por ler!
